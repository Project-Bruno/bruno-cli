# Usage
<!-- usage -->
```sh-session
$ npm install -g bruno-cli
$ bruno COMMAND
running command...
$ bruno (-v|--version|version)
bruno-cli/0.0.0 linux-x64 node-v12.19.0
$ bruno --help [COMMAND]
USAGE
  $ bruno COMMAND
...
```
